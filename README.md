# Resume Website (Static)

This is a simple static resume site (HTML/CSS/JS). No build tools required.

## Run locally
Just open `index.html` in a browser, OR:
- VS Code → install "Live Server" extension → right click `index.html` → "Open with Live Server"

## Free hosting options (no cost)
### Option A — GitHub Pages (recommended)
1. Create a GitHub account (if you don't have one).
2. Create a new repo named: `YOURUSERNAME.github.io`
3. Upload these files (index.html, styles.css, script.js, /assets).
4. Go to repo Settings → Pages → Deploy from branch → `main` / root
5. Your site will be live at: https://YOURUSERNAME.github.io

### Option B — Netlify (fast drag-and-drop)
1. Go to Netlify → Add new site → Deploy manually
2. Drag the folder containing these files
3. You'll get a `*.netlify.app` URL.

### Option C — Vercel (also easy)
1. Import the GitHub repo in Vercel
2. Framework preset: "Other"
3. Deploy → you get a `*.vercel.app` URL.
